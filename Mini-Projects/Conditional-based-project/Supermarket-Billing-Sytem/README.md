#  Supermarket Billing System (Python)

## Description
This is a simple Supermarket Billing System built using Python.  
The project calculates the total bill based on item price, GST, membership discount, and home delivery charges.

## Features
- Select product item
- Calculate GST based on product type
- Membership discount (Gold / Silver)
- Home delivery charge
- Quantity validation
- Generates final bill

## Concepts Used
- Variables
- Conditional statements (if, elif, else)
- User input
- Arithmetic operations

- ## Example Items
- Sugar
- Chocolate
- Green Tea
- Mobile
- Laptop
