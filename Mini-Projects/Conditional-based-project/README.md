# Conditional Based Python Projects

## Description
This folder contains Python mini projects built while learning conditional statements (if, elif, else) and basic programming logic.

These projects help in understanding how decision making works in real-world programs.

## Projects Included

1. ATM System
   - Simple ATM simulation with login, balance check, deposit and withdraw.

2. Electricity Bill Calculator
   - Calculates electricity bill based on unit consumption and billing rules.

3. Kalyan Jewellers Billing System
   - Calculates gold price, making charges and applies discounts based on conditions.

4. Supermarket Billing System
   - Generates a bill based on item price, GST, membership discount and delivery charges.

5. Mini Python Calculator
   - Performs basic operations like addition, subtraction, multiplication and division.

6. Number Guessing Game
   - A small game where the user guesses a random number between 1 and 100.

## Concepts Used
- Variables
- User Input
- Conditional Statements (if / elif / else)
- While Loop
- Arithmetic Operations

## Purpose
These projects were created for practice while learning Python fundamentals and building logical thinking.
