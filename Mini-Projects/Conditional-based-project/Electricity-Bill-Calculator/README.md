# Electricity Bill Calculator (Python)

This is a console-based Python mini project that calculates the electricity bill based on unit consumption.

The program includes:
- Unit slab billing
- Fixed meter charges
- Peak hour extra charges
- Surcharge for high usage
- Discount for low consumption
- Late payment penalty

Finally, it generates a formatted bill with the total payable amount.

## Purpose
This project was created to practice:
- Conditional statements
- Arithmetic calculations
- Slab based billing logic
- User input handling
