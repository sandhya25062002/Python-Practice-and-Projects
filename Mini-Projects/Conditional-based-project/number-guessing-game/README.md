# Number Guessing Game

This is a simple Python mini project that I built while learning conditional statements.

The computer generates a random number between 1 and 100 and the user has to guess the correct number. 
The program gives hints like "Too High" or "Too Low" until the correct number is guessed.

## Features
- Random number generation
- User input
- While loop
- Conditional statements
- Attempt counter

## How to Run

1. Run the Python file
2. Enter a number between 1 and 100
3. Follow the hints (Too High / Too Low)
4. Guess the correct number to win
