## Electricity Bill Calculator (Python)

This is a console-based Python mini project that calculates the monthly electricity bill based on unit consumption.

The program calculates:

Unit-wise slab billing

Fixed meter charges

Peak-hour extra charges

Surcharge (if units exceed limit)

Discount for low consumption

Late payment penalty

It then generates a final formatted electricity bill with total payable amount and consumption type.

## Purpose

To practice:

Conditional statements

Arithmetic calculations

Slab-based billing logic

Real-world problem solving

User input handling
