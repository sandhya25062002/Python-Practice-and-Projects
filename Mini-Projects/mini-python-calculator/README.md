# Mini Calculator (Python)

This is my first Python calculator project.
It is a simple command-line calculator that performs basic mathematical operations using user input.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Exit option
- Error handling for invalid input
- Prevents division by zero

## Concepts Used
- Python while loop
- if-elif-else conditions
- try-except (error handling)
- user input
- basic arithmetic operations
